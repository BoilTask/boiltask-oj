## 描述

现在给你一个由小写字母组成字符串，要你找出字符串中出现次数最多的字母，如果出现次数最多字母有多个那
么输出最小的那个。

## 输入

第一行输入一个正整数T（0<T<25）
随后T行输入一个字符串s,S长度小于1010。
## 输出

每组数据输出占一行，输出出现次数最多的字符；

## 样例输入

```
abcd
bbaa
jsdhfjkshdfjksahdfjkhsajkf
```

## 样例输出

```
a
a
j
```

## 题目描述

按递增顺序依次列出所有分母为40，分子小于40的最简分数。

## 输入

无输入

## 输出

分数之间用逗号分开(含最末逗号)

## 样例输出

> 以下仅用于展示格式，数值并不符合要求

```
1/2,2/3,4/5,
```

## 提示

对分子采用穷举法，利用最大公约数的方法，判断分子与40是否构成真分数。

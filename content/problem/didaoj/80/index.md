## 问题描述

正在下五子棋的KACA突发奇想，能不能创造一种二子棋。

很明显如果和五子棋一样是二字连珠便算赢，那么先手便可以获胜。所以KACA便设计，黑白双方交替落子，不允许同色的二子连珠，因为棋盘是有限的，所以哪一方没办法下子便判为输。

![20161109165804_85452.png](https://r2-oj.didapipa.com/problem/80/1761667116_53daa498-c717-4c9a-b1a4-6d469431070a)


如图，若在中间落子，则周围8格都不可以再落自己的子了

(因为必然连成两个)。

于是KACA便以这个规则与PIPA对弈，然而他发现游戏的胜负似乎与是否先手有关系，所以他想要更深一步研究一下，如果双方都希望自己嬴，那么最终哪一方能获胜。

## 输入

多组测试数据。

每一组有一行，为两个以空格隔开的数字n,m(0&lt;n,m&lt;100)分别代表棋盘的行数和列数。

## 输出

对于每一组数据，在两方都不失误的情况下，若先手方嬴，则输出“Bwin”，若后手方嬴，则输出“Ewin”。

## 样例输入

```
1 1
1 2

```

## 样例输出

```
Bwin
Ewin

```

## Description
Fibonacci数列，定义如下：  
f(1)=f(2)=1  
f(n)=f(n-1)+f(n-2) n>=3。  
计算第n项Fibonacci数值。

## Input
输入第一行为一个整数N，接下来N行为整数Pi（1<=Pi<=1000）。

## Output

输出为N行，每行为对应的f(Pi)。

## Sample Input

```
5
1
2
3
4
5
```

## Sample Output

```
1
1
2
3
5
```

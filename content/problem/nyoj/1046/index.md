## 题目描述

给你一个非零整数，让你求这个数的n次方，每次相乘的结果可以在后面使用，求至少需要多少次乘。如24：2\*2=22（第一次乘），22\*22\=24（第二次乘），所以最少共2次；

## 输入描述

第一行m表示有m(1<=m<=100)组测试数据；  
每一组测试数据有一整数n（0<n<=10000）;

## 输出描述

输出每组测试数据所需次数s;

## 用例

### 用例输入

```
3
2
3
4
```
  

### 用例输出

```
1
2
2
```

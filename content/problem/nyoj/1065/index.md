## 题目描述

  

大家都知道阶乘这个概念，举个简单的例子：5！=1\*2\*3\*4\*5.现在我们引入一种新的阶乘概念，将原来的每个数相乘变为i不大于n的所有奇数相乘例如：5!!=1\*3\*5.现在明白现在这种阶乘的意思了吧！

现在你的任务是求出1!!+2!!......+n!!的正确值(n<=20)

## 输入描述

第一行输入一个a(a<=20)，代表共有a组测试数据  
接下来a行各行输入一个n.

## 输出描述

各行输出结果一个整数R表示1!!+2!!......+n!!的正确值

## 用例

### 用例输入

```
2
3
5
```  

### 用例输出

```
5
23
```

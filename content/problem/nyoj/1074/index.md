## 题目描述

很多小学生在学习加法时，发现“进位”特别容易出错。你的任务是计算两个三位数在相加时需要多少次进位。你编制的程序应当可以连续处理多组数据，直到读到两个0（这是输入结束标记）。

## 输入描述

输入两个正整数m,n.(m,n,都是三位数)

## 输出描述

输出m,n,相加时需要进位多少次。

## 用例

### 用例输入

```
123 456
555 555
123 594
0 0
```  

### 用例输出

```
0
3 
1

```

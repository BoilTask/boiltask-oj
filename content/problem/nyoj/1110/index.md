## 题目描述

编写一个C程序，实现两个分数的加减法

## 输入描述

输入包含多行数据  
每行数据是一个字符串，格式是"a/boc/d"。  
其中a, b, c, d是一个0-9的整数。o是运算符"+"或者"-"。  
  
数据以EOF结束  
输入数据保证合法

## 输出描述

对于输入数据的每一行输出两个分数的运算结果。  
注意结果应符合书写习惯，没有多余的符号、分子、分母，并且化简至最简分数

## 用例

### 用例输入

```
1/8+3/8
1/4-1/2
1/3-1/3
```  

### 用例输出

```
1/2
-1/4
0
```

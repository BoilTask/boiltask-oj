## 题目描述

给定 n 堆石子，两位玩家轮流操作，每次操作可以从任意一堆石子中拿走任意数量的石子（可以拿完，但不能不拿），最后无法进行操作的人视为失败。 问如果两人都采用最优策略，先手是否必胜。

## 输入描述

第一行包含整数 n。  
  
第二行包含 n 个数字，其中第 i 个数字表示第 i 堆石子的数量。

## 输出描述

如果先手方必胜，则输出 Yes。  
  
否则，输出 No。

## 用例

### 用例输入

```
2
2 3
```
  

### 用例输出

```
Yes
```

## 提示

1≤n≤30, 1≤每堆石子数≤100 这是位运算的题目，想想取筷子

## 题目描述

已知鸡和兔的总数量为n,总腿数为m。输入n和m,依次输出鸡和兔的数目，如果无解，则输出“No answer”(不要引号)。

## 输入格式

第一行输入一个数据a,代表接下来共有几组数据，在接下来的(a&lt;10)<br />a行里，每行都有一个n和m.(0&lt;m,n&lt;100)

## 输出格式

输出鸡兔的个数，或者No answer

## 样例

<h3>样例输入</h3>

```
2
14 32
10 16


```
<h3>样例输出</h3>

```
12 2
No answer

```

## 题目描述

请判断一个数是不是水仙花数。   
其中水仙花数定义各个位数立方和等于它本身的三位数。

## 输入格式

有多组测试数据，每组测试数据以包含一个整数$n(100<=n<1000)$
输入$0$表示程序输入结束。

## 输出格式

如果n是水仙花数就输出Yes
否则输出No

## 样例

### 样例输入
```
153
154
0
```
### 样例输出
```
Yes
No
```

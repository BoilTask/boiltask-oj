## 题目描述

大家一定都做过A+B吧，立刻AC吧。

## 输入

多组输入：

每组输入一个字符串"A+B" 。（A，B是正整数，且A+B < 231）

## 输出

对每组输入，输出A+B的结果。

## 样例输入

```
1+2
3+4
```

## 样例输出

```
3
7
```

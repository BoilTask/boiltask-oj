## 题目描述

JS看最近GG在疯狂的学习数学，于是想看看GG数学学习的成果。

JS：GG巨，听说最近你在学数学？

GG：是啊。

JS：那我给你出个数学题吧0.0！！！

GG：好啊。

JS：给你一个整数N，那么是否存在最小正整数K，使得N%(K+1)=0?

GG：。。。。。。。

GG被这个题难到了。聪明的ACMer，你能帮帮他么？

## 输入

本题为单实例测试

输入一个N，代表这个整数。（2=<N<=2000000000）

## 输出

输出占一行。输出最小的正整数K。

## 样例输入

```
5
```

## 样例输出

```
4
```

## 题目描述

给你N个整数，统计这些数中大于X的数的个数。

## 输入

输入分两行。

第一行输入N X。（N X的意思如题目描述，N<=10）

第二行输入N个整数。

## 输出

输出一个数，代表N个数中大于X的数的个数。

## 样例输入

```
5 3
1 3 4 5 8
```

## 样例输出

```
3
```

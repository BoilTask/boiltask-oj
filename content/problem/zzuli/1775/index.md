## 题目描述

最近和尚因为一件事特烦恼。有一位施主向和尚询问一个正整数是不是素数的问题。你愿意帮助和尚解决这个烦恼吗？

## 输入

输入数据有多组(以EOF结束)。每组数据首先输入一个整数n(n<100). 然后有n个正整数。(每个正整数<= 100000)。

## 输出

对于每个正整数，如果是素数则输出"Yes",如果不是输出"No"

## 样例输入

```
2
3
6

3
5
8
13
```

## 样例输出

```
Yes
No
Yes
No
Yes
```

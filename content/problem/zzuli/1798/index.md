## 题目描述

戴姆勒最近比较喜欢平方数，自称掌握所有关于平方数的题。

灿灿听到后就不高兴了，立马要考考他。

聪明的灿灿要用立方数来难倒他。

他给戴姆勒一个数字n，问他有从1到n有多少个完全立方数（1，8，27，，，，）？

戴姆勒瞬间傻眼了（好可怜），于是请求你来帮助他，你能帮助他吗？

## 输入

输入第一行为一个整数T（0<T<=1000），表示有T组测试实例。

每组测试实例有一个数字n（1<=n<=1018)。

## 输出

请输出从1到n中立方数的个数；

## 样例输入

```
3
1
2
8
```

## 样例输出

```
1
1
2
```

## 提示

如果使用了pow函数，请尝试加不同大小的小数修正来修正精度。

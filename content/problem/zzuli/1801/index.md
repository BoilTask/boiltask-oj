## 题目描述

xue姐喜欢动物，于是养了好多动物，动物的食物成了xue姐的最大难题。

xue姐共有n个动物，每个动物有一定的饥饿值，于是xue姐买了n个食物。

每个食物有一定的能量值，每个动物吃一个食物，该食物的能量值大于等于该动物的饥饿值时才算吃饱（每个食物只能一只动物吃，不能分开）。

问xue姐的动物们能否全部都吃饱？

## 输入

第一行为数字T，表示有T（1<=T<=100）组测试数据。每组测试数据第一行为数字n（1<=n<=100），

第二行为n个数字表示每只动物的饥饿值，

第三行为n个数字表示每个食物的能量值。

## 输出

xue姐的动物们能吃饱输出“YES”，不能输出“NO”（不加引号）。

## 样例输入

```
1
3
1 2 3
4 5 6
```

## 样例输出

```
YES
```

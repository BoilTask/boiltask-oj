## 题目描述

众所周知，LT是101的一号人物，顶级的存在，但是同一宿舍的DX和WT对此表示不服，他们决定出一道难题来考考LT。

两个人绞尽脑汁，终于想到一道绝世难题。他们会给LT一个任意的正整数n，让LT立刻求出1-n个数字的和。果然LT无法解决这个问题，于是他向你们求助，你们能帮帮他吗？

## 输入

输入第一行是一个T（1<=T<=10），代表T组测试数据。

接下来T行， 每行一个正整数n（$1<=n<=10^9$）。

## 输出

每行一个数字，代表1-n的和。

## 样例输入

```
2
1
3
```

## 样例输出

```
1
6
```

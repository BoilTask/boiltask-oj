## 题目描述

擅于解密游戏的YZ同学在玩过101种解密游戏后是不虚任何解密游戏的，他认为好的游戏应该大家伙一起玩（我才不玩勒）所以我把今天他给我的解密游戏送给大家玩要，，，当然这个很简单的。游戏规则如下：

对于给定的两个字符串看看他们是否完全匹配，字符的匹配规则是这样的，Y与Z，V与I，D与X，L与T分别匹配。当然字符串1由Y，Y，D，L构成，字符串2由Z，T，X构成，不会有其他多余字符。如果完全匹配的话YZ同学就可以得到巨额的金币，去解锁其他好玩的游戏。

## 输入

第一行一个整数T，代表有多少组数据。

第二行为一个整数N，代表两个字符串的长度。

第三行分别是两个长度为N的字符串。1 <= T <= 10, 1 <= N <= 10000.

## 输出

对于每组数据。如果完全匹配的话输出"YZ同学得到了许多金币^_^"否则输出"YZ同学什么都没拿到T_T"。

## 样例输入

```
2
4
YWDL ZTXT
6
YWWDDL ZTXTXX
```

## 样例输出

```
YZ同学得到了许多金币^_^
YZ同学什么都没拿到T_T
```
